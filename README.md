# 2048 Game

A browser-based implementation of the classic 2048 puzzle game.  
The player moves tiles on a 4x4 board using arrow keys, combines matching numbers, increases the score, and tries to reach the 2048 tile.

## Live Preview

[DEMO LINK](https://horbenkoOleksandr.github.io/2048-game/)

## Technologies Used

- HTML5
- SCSS
- JavaScript
- ES Modules
- Gulp

## Getting Started

```bash
git clone https://github.com/horbenkoOleksandr/2048-game.git
